# 命盘推演引擎 (MingPan Engine)

> 🎯 **命理推演仿真系统** - 用现代技术重构传统命理学，实现动态情境推演

## ✨ 核心特性

### 五大计算模块
1. **四柱八字引擎** - 排盘、十神、大运流年、旺衰格局
2. **紫微斗数引擎** - 十二宫、主星、四化飞星、大限流年
3. **梅花易数引擎** - 时间/数字/方位起卦、体用生克、应期推算
4. **人际关系耦合** - 合婚/合盘、五行互补、流年共振
5. **风水环境因子** - 八宅风水、玄空飞星、流年紫白

### OASIS动态推演
- **多智能体仿真**：每个人 = Agent，命盘 = 属性基线
- **环境系统**：流年大运 = 全局变量，风水方位 = 局部变量
- **推演输出**：概率云 + 运势热力图 + 关键决策建议

### 知识库RAG
- 天纪紫微、紫微斗数知识汇集
- 周易解析（64卦×6爻）
- 四柱八字干支命理
- LLM个性化解读生成

## 🛠️ 技术栈

### 后端
- **语言**: Python 3.11+
- **框架**: FastAPI
- **计算**: NumPy, SciPy, Skyfield
- **AI/ML**: LangChain, vLLM, Milvus
- **任务队列**: Celery + Redis

### 前端
- **框架**: React 18 + TypeScript
- **UI**: Ant Design / Arco Design
- **图表**: ECharts, D3.js, SVG/Canvas
- **状态**: Zustand

### 数据库
- **主库**: PostgreSQL 15+
- **缓存**: Redis 7+
- **向量库**: Milvus 2.x
- **图库**: Neo4j 5.x (可选)

### 部署
- **容器**: Docker + Docker Compose
- **编排**: Kubernetes
- **CI/CD**: GitHub Actions

## 🚀 快速开始

### 环境准备
```bash
# 克隆项目
git clone <repository-url>
cd MingPanEngine

# Python环境
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 前端环境
cd frontend
npm install
```

### 启动服务
```bash
# 开发环境
docker-compose up -d

# 或手动启动
cd backend/services/bazi-service
uvicorn main:app --port 8001 --reload
```

### 运行测试
```bash
# 后端测试
pytest tests/

# 前端测试
cd frontend
npm test
```
