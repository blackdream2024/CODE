# SenseDraw 版本说明

## v1.0.0 (2026-05-13)

### 🎉 首次发布

这是 SenseDraw 的首个正式版本，基于 SenseNova AI 提供智能系统图生成功能。

---

### ✨ 新增功能

#### 核心功能
- **文生图模式**：输入文字描述，AI 自动生成系统架构图
- **图生图模式**：上传参考图片，AI 分析后精准还原或创意扩展
- **文档分析**：上传文档（TXT/DOC/DOCX/PDF）自动生成架构图描述
- **提示词增强**：AI 优化描述文本，提升生成质量
- **识图功能**：AI 识别图片内容，提取描述信息

#### 场景模板
- **系统架构图**：科技蓝白配色，模块化设计，适合技术方案
- **国防态势图**：深绿深蓝配色，指挥大屏风格，适合军事安全
- **应急流程图**：政务蓝配色，流程节点清晰，适合应急管理
- **网络拓扑图**：深色背景，分层架构，适合网络部署
- **信息图/简报**：现代简洁风格，数据可视化，适合商业展示
- **自由模式**：无模板限制，AI 自由发挥，适合创意设计

#### 高级选项
- **多种图片尺寸**：支持 16:9、1:1、3:2、2:3、4:3、3:4 等比例
- **输出格式**：支持 PNG 图片和 SVG 矢量图
- **AI 分析过程**：可查看 AI 理解和生成过程

#### 管理功能
- **历史记录**：保存所有生成记录，支持查看和管理
- **API 使用统计**：实时显示 API 调用次数和剩余配额
- **今日生成统计**：统计今日生成数量

---

### 🛠️ 技术特性

#### 前端技术
- **响应式设计**：适配不同屏幕尺寸
- **现代 UI**：简洁美观的用户界面
- **实时预览**：图片上传即时预览
- **拖拽上传**：支持拖拽文件上传

#### 后端技术
- **Node.js + Express**：高性能后端服务
- **SenseNova API**：集成最新 AI 图像生成能力
- **PDF.js**：支持 PDF 文档解析
- **文件缓存**：优化图片加载速度

#### 安全特性
- **内容安全检查**：自动过滤敏感内容
- **错误处理**：友好的错误提示
- **输入验证**：防止恶意输入

---

### 📦 安装与部署

#### 系统要求
- Windows 10/11
- Node.js 18.0+
- 稳定的互联网连接

#### 安装方式
1. **一键安装**：使用 `install.bat` 自动安装依赖
2. **手动安装**：通过 npm 手动安装

#### 配置要求
- SenseNova API Key（从 SenseNova 平台获取）
