# SenseDraw 快速入门指南

## 5分钟快速上手

### 第一步：安装启动

1. 解压 `SenseDraw-1.0.0.zip` 到任意目录
2. 双击 `install.bat` 安装依赖
3. 双击 `start.bat` 启动服务
4. 浏览器打开 `http://localhost:3456`

### 第二步：配置 API Key

编辑 `backend/u1-pipeline-server.js`，修改第 8 行：
```javascript
const API_KEY = '你的SenseNova API Key';
```

---

## 核心功能速查

### 文生图（最快方式）

1. 选择"文生图"模式
2. 输入描述，例如：
   ```
   电商系统架构，包含用户端、API网关、订单服务、支付服务、数据库
   ```
3. 点击"生成图像"
4. 等待 10-30 秒

### 图生图（基于参考图）

1. 选择"图生图"模式
2. 上传参考图片
3. 选择子模式（推荐"精准还原"）
4. 输入优化要求
5. 点击"生成图像"

### 文档分析（自动生成）

1. 选择"文生图"模式
2. 点击"上传文档自动生成描述"
3. 选择 TXT/DOC/DOCX/PDF 文件
4. 等待 AI 分析
5. 点击"采纳此提示词"

## 提升生成质量的技巧

### 描述公式

```
[系统类型] + [核心组件] + [连接关系] + [技术栈] + [风格要求]
```

**示例**：
```
电商系统架构，包含用户端、API网关、订单服务、支付服务、库存服务，
通过REST API连接，使用MySQL数据库和Redis缓存，科技蓝配色，模块化布局
```

### 快捷操作

- **提示词增强**：点击"🧠 提示词增强"让AI优化描述
- **识图功能**：上传图片后点击"🔍 识图"提取描述
- **历史记录**：侧边栏"📁 历史记录"查看所有生成

## 常见问题速解

| 问题 | 解决方案 |
|------|---------|
| 生成慢（>30秒） | 检查网络，简化描述 |
| 内容安全错误 | 避免敏感词汇 |
| 效果不理想 | 使用提示词增强，选择合适模板 |
| 端口占用 | 关闭占用3456端口的程序 |
| 文档分析失败 | 检查格式和大小限制 |
