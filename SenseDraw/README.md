## 功能特性

- 🎨 **文生图**：输入文字描述，生成系统架构图/态势图/流程图
- 🖼️ **图生图**：上传参考图，AI 理解后重新生成优化版本
- 📊 **6种专业场景**：系统架构、国防态势、应急流程、网络拓扑、信息图、自由模式
- 📋 **历史管理**：保存生成记录，支持重新编辑和迭代

## 快速开始

### 前置要求

- Node.js 16+ 
- SenseNova API Key

### 安装步骤

1. **克隆或下载项目**
   ```bash
   # 项目已包含在 SenseDraw 目录中
   cd SenseDraw
   ```

2. **配置 API Key**
   
   打开 `backend/u1-pipeline-server.js`，找到第 8 行：
   ```javascript
   const API_KEY = '你的API Key';
   ```
   替换为你的 SenseNova API Key。

3. **安装后端依赖**
   ```bash
   cd backend
   npm install
   ```

4. **启动后端服务**
   ```bash
   npm start
   ```
   
   看到以下信息表示启动成功：
   ```
   🎨 SenseDraw Backend Server
   服务地址: http://localhost:3456
   ```

5. **打开前端页面**
   
   直接用浏览器打开 `frontend/index.html` 即可使用。

## 使用指南

### 文生图模式

1. 选择「文生图」模式
2. 选择场景模板（如系统架构图）
3. 输入描述，例如：
   > 一个电商系统架构，包含用户端、API网关、订单服务、支付服务、库存服务、数据库层
4. 点击「生成图像」
5. 等待 AI 生成并查看结果

### 图生图模式

1. 选择「图生图」模式
2. 上传参考图片或输入图片 URL
3. 选择场景模板
4. 输入优化要求，例如：
   > 保持整体结构，优化配色为科技蓝，增加更多细节
5. 点击「生成图像」
6. 查看优化后的结果

### 场景模板说明

| 场景 | 描述 | 适用场景 |
|------|------|----------|
| 🏗️ 系统架构图 | 科技蓝白，模块化设计 | 技术方案、系统设计 |
| 🛡️ 国防态势图 | 深绿深蓝，指挥大屏风格 | 军事、国防、安全 |

| 🎨 自由模式 | 无模板，直接生成 | 创意设计、自定义需求 |

## API 接口

### 健康检查
```
GET /health
```

### 获取场景列表
```
GET /api/scenes
```

### 生成图像
```
POST /api/generate
```

**请求参数：**
```json
{
  "mode": "text_to_image | image_to_image",
  "scene": "system_arch | defense_posture | emergency_flow | network_topo | infographic | raw",
  "prompt": "描述文本",
  "image_url": "参考图片URL（图生图模式必填）",
  "size": "1024x1024 | 1536x1024 | 1024x1536",
  "n": 1
}
```

**响应示例：**
```json
{
  "success": true,
  "mode": "text_to_image",
  "images": ["https://..."],
  "prompt": "完整prompt",
  "meta": {
    "model": "sensenova-u1-fast",
    "scene": "system_arch",
    "size": "1024x1024",
    "timestamp": "2026-05-12T12:00:00.000Z"
  }
}
```

## 常见问题

### Q: 生成图片很慢？
A: 图片生成通常需要 10-30 秒，请耐心等待。如果超过 60 秒无响应，请检查网络连接。

### Q: 支持哪些图片尺寸？
A: 支持以下尺寸：
- 1024x1024 (1:1 方形)
- 1536x1024 (3:2 横版)
- 1024x1536 (2:3 竖版)

### Q: 如何获取 API Key？
A: 访问 [SenseNova 平台](https://platform.sensenova.cn/) 注册账号并获取 API Key。
