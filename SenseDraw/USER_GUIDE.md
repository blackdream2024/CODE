## 软件简介

**SenseDraw** 是一款基于 SenseNova AI 的智能系统图生成工具，专为技术人员、架构师、设计师打造。它能够将文字描述或参考图片转化为专业的系统架构图、流程图、态势图等多种图表。

### 核心特性

- **文生图**：输入文字描述，AI 自动生成系统架构图
- **图生图**：上传参考图片，AI 分析后精准还原或创意扩展
- **文档分析**：上传文档自动提取关键信息生成架构图
- **智能增强**：AI 优化描述文本，提升生成质量
- **专业模板**：6种预设场景，满足不同领域需求
- **历史管理**：保存生成记录，支持迭代优化

## 系统要求

### 硬件要求
- **处理器**：Intel Core i3 或同等性能以上
- **内存**：4GB RAM 以上
- **硬盘空间**：500MB 可用空间
- **网络**：需要稳定的互联网连接

### 软件要求
- **操作系统**：Windows 10/11
- **浏览器**：Chrome 90+、Firefox 88+、Edge 90+（推荐使用最新版本）
- **运行环境**：Node.js 18.0 或更高版本

### API 要求
- **SenseNova API Key**：需要从 [SenseNova 平台](https://platform.sensenova.cn/) 获取

## 安装与启动

### 方法一：使用安装包（推荐）

1. **获取安装包**
   - 下载 `SenseDraw-1.0.0.zip` 文件

2. **解压文件**
   - 将 ZIP 文件解压到任意目录（建议路径不含中文）

3. **一键安装**
   - 双击运行 `install.bat`
   - 等待依赖安装完成

4. **启动程序**
   - 双击运行 `start.bat`
   - 程序将在 `http://localhost:3456` 启动

5. **访问界面**
   - 在浏览器中打开 `http://localhost:3456`

### 方法二：手动安装

1. **安装 Node.js**
   - 访问 https://nodejs.org/
   - 下载并安装 LTS 版本

2. **配置 API Key**
   - 打开 `backend/u1-pipeline-server.js`
   - 找到第 8 行：`const API_KEY = '你的API Key';`
   - 替换为你的 SenseNova API Key

3. **安装后端依赖**
   ```bash
   cd backend
   npm install
   ```

4. **启动服务**
   ```bash
   npm start
   ```

5. **访问界面**
   - 打开浏览器访问 `http://localhost:3456`

## 详细使用指南

### 文生图模式

**适用场景**：从零开始创建系统架构图

**操作步骤**：

1. **选择模式**
   - 在"创建任务"页面，选择"文生图"模式

2. **选择场景模板**（可选）
   - 根据需求选择合适的场景模板
   - 不同模板会影响生成风格

3. **输入描述**
   - 在文本框中详细描述你想要的架构图
   - **描述技巧**：
     - 明确系统组件：如"用户端、API网关、订单服务"
     - 说明连接关系：如"通过REST API连接"
     - 指定技术栈：如"使用MySQL数据库"
     - 描述整体风格：如"科技蓝配色"

   **示例描述**：
   ```
   一个电商系统架构，包含以下组件：
   - 用户端：Web前端和移动端App
   - API网关：Nginx负载均衡
   - 微服务层：订单服务、支付服务、库存服务、用户服务
   - 数据层：MySQL主从集群、Redis缓存
   - 消息队列：RabbitMQ
   - 监控：Prometheus + Grafana
   
   要求：科技蓝配色，模块化布局，清晰的连接线
   ```

4. **高级选项**（可选）
   - 调整图片尺寸
   - 选择输出格式（PNG/SVG）
   - 显示AI分析过程

5. **生成图像**
   - 点击"生成图像"按钮
   - 等待10-30秒

6. **查看结果**
   - 查看生成的架构图
   - 可下载或重新生成

---

### 图生图模式

**适用场景**：基于现有图片优化或重新生成

**操作步骤**：

1. **选择模式**
   - 选择"图生图"模式

2. **选择图生图子模式**
   - **精准还原**：只优化效果，不增减内容
   - **延展发挥**：基于原图创意扩展
   - **结构锁定**：坐标精确，只优化视觉风格

3. **上传参考图片**
   - **方式一**：点击上传区域选择图片文件
   - **方式二**：拖拽图片到上传区域
   - **方式三**：输入图片URL

4. **识图**（可选）
   - 点击"识图"按钮
   - AI 会分析图片内容并生成描述
   - 可对描述进行文字增强

5. **输入优化要求**
   - 描述你希望如何优化参考图片
   - **示例**：
     ```
     保持原有架构结构，优化以下方面：
     1. 配色改为科技蓝主题
     2. 增加更多技术细节标注
     3. 优化布局，使层次更清晰
     4. 添加图标和装饰元素
     ```

6. **生成图像**
   - 点击"生成图像"按钮
   - 等待AI处理

7. **查看结果**
   - 对比原图和生成图
   - 可查看AI分析过程

### 文档分析功能

**适用场景**：从文档中提取信息生成架构图

**支持格式**：TXT、DOC、DOCX、PDF

**操作步骤**：

1. **进入文生图模式**
   - 文档分析功能在文生图模式下使用

2. **上传文档**
   - 点击"上传文档自动生成描述"
   - 选择文档文件
   - **注意**：最大支持 30,000 字符（约 15,000 汉字）

3. **AI 分析**
   - 系统自动读取文档内容
   - AI 分析文档结构和关键信息
   - 生成架构图描述

4. **查看分析结果**
   - 弹窗显示文档摘要
   - 显示生成的提示词
   - 可选择"采纳此提示词"

5. **生成图像**
   - 使用生成的提示词创建架构图

**文档分析技巧**：
- 文档结构越清晰，分析效果越好
- 包含技术术语的文档分析更准确
- 长文档会提取关键信息，不会全部使用

---

### 提示词增强

**适用场景**：优化描述文本，提升生成质量

**操作步骤**：

1. **输入基础描述**
   - 在文本框中输入基本描述

2. **点击提示词增强**
   - 点击"🧠 提示词增强"按钮
   - AI 会分析并优化描述

3. **查看增强结果**
   - 弹窗显示优化后的描述
   - 显示使用的AI模型

4. **选择操作**
   - **采用此描述**：使用优化后的描述
   - **重新生成**：再次优化
   - **取消**：使用原始描述

**增强效果**：
- 添加更多技术细节
- 优化语言表达
- 补充缺失的组件
- 明确连接关系

---

### 识图功能

**适用场景**：分析现有图片，提取描述信息

**操作步骤**：

1. **上传图片**
   - 在图生图模式下上传参考图片

2. **点击识图**
   - 点击"🔍 识图"按钮
   - AI 分析图片内容

3. **查看识别结果**
   - 弹窗显示图片内容描述
   - 可进行文字增强

4. **应用描述**
   - 点击"采用此描述"
   - 描述会填入文本框

**识别内容**：
- 图片中的组件和元素
- 组件之间的连接关系
- 整体架构风格
- 颜色和布局特点

| 模板名称 | 风格特点 | 适用场景 | 配色方案 |
|---------|---------|---------|---------|
| 🏗️ 系统架构图 | 科技蓝白，模块化设计 | 技术方案、系统设计、架构评审 | 蓝色主调，白色背景 |
| 🛡️ 国防态势图 | 深绿深蓝，指挥大屏风格 | 军事、国防、安全监控 | 深绿/深蓝，暗色背景 |

| 🌐 网络拓扑图 | 深色背景，分层架构 | 网络架构、IT基础设施、云部署 | 深色背景，亮色节点 |

| 🎨 自由模式 | 无模板，直接生成 | 创意设计、自定义需求、实验性生成 | 由AI自由发挥 |

**选择建议**：
- 技术架构 → 系统架构图
- 安全相关 → 国防态势图
- 流程管理 → 应急流程图
- 网络部署 → 网络拓扑图
- 商业展示 → 信息图/简报

## 常见问题解答

### Q1: 生成图片很慢怎么办？

**A**: 图片生成通常需要 10-30 秒，如果超过 60 秒无响应：
1. 检查网络连接
2. 确认 API Key 是否有效
3. 尝试简化描述文本
4. 刷新页面重试

### Q2: 提示"内容安全检查未通过"怎么办？

**A**: 这是因为描述中包含了敏感内容：
1. 避免使用暴力、色情、政治敏感词汇
2. 使用专业术语替代口语化表达
3. 简化描述，保留核心信息
4. 尝试使用其他场景模板

### Q3: 生成的图片不符合预期怎么办？

**A**: 尝试以下优化方法：
1. **使用提示词增强**：让AI优化描述
2. **添加更多细节**：明确组件、连接关系、技术栈
3. **选择合适的模板**：不同模板风格差异很大
4. **调整图片尺寸**：选择适合的宽高比
5. **使用图生图模式**：基于参考图片生成

### Q4: 如何获取 API Key？

**A**: 
1. 访问 [SenseNova 平台](https://platform.sensenova.cn/)
2. 注册账号并登录
3. 在控制台获取 API Key
4. 将 Key 填入配置文件

### Q5: 支持哪些图片格式？

**A**: 
- **上传格式**：JPG、PNG、GIF、WebP
- **输出格式**：PNG、SVG
- **建议**：上传 PNG 格式，质量最高

### Q6: 文档分析失败怎么办？

**A**: 
1. 检查文档格式是否支持（TXT、DOC、DOCX、PDF）
2. 确认文档大小不超过 30,000 字符
3. 检查文档是否加密或损坏
4. 尝试转换为 TXT 格式后重试

### Q7: 如何提高生成质量？

**A**: 
1. **详细描述**：包含组件、关系、技术栈
2. **使用模板**：选择最接近的场景模板
3. **提示词增强**：让AI优化描述
4. **多次迭代**：基于结果调整描述
5. **参考图片**：使用图生图模式

### Q8: 端口被占用怎么办？

**A**: 
1. 关闭占用 3456 端口的程序
2. 或修改 `backend/u1-pipeline-server.js` 中的 PORT 变量
3. 重启后端服务

### Q9: 如何备份历史记录？

**A**: 
1. 定期导出重要图片
2. 使用浏览器书签保存常用描述
3. 记录重要的提示词文本

### Q10: 支持团队协作吗？

**A**: 当前版本为本地单机版，不支持：
1. 多用户同时使用
2. 云端同步历史记录
3. 团队共享配置
