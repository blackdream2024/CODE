## 🔍 文档搜索提示

### 在浏览器中搜索
1. 打开文档页面
2. 按 `Ctrl + F` 打开搜索框
3. 输入关键词搜索

### 常用搜索关键词

- **安装**：Node.js、npm、端口、API Key
- **功能**：文生图、图生图、文档分析、提示词增强
- **问题**：失败、错误、慢、无法访问
- **配置**：尺寸、格式、模板、场景
