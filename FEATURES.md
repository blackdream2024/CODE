# 健身房划算寻找功能说明

## 🎯 核心功能

### 1. 智能套餐比价系统

#### 详情页增强功能

**最划算套餐自动计算**
- 系统自动分析所有套餐，计算性价比得分
- 综合考虑：折扣力度、月均成本、服务数量、会员时长
- 自动标记"🏆 最划算"套餐

**对比模式**
- 点击"📊 开启对比模式"按钮
- 显示每个套餐的详细对比信息：
  - 月均成本（折算为月费用，便于横向比较）
  - 折扣力度（百分比显示）
  - 服务数量（权益项数）
  - 节省金额（原价 - 现价）

**性价比标签**
- 🏆 最划算：综合得分最高的套餐
- 💰 超值：月均成本 < 100元
- 👍 推荐：月均成本 < 200元
- 📋 标准：其他套餐

### 2. 首页智能推荐

**划算提示横幅**
- 顶部显示"已为您智能筛选附近最划算健身房"

**列表卡片增强**

**排序逻辑**
- 健身房按性价比排序
- 综合考虑：价格、距离、评分、折扣力度

## 📊 算法说明

### 性价比得分计算公式

```javascript
valueScore = (折扣比例 × 100) + (服务数量 × 5) + 长期会员奖励
```

**参数说明：**
- **折扣比例**：(原价 - 现价) / 原价
- **服务数量**：套餐包含的权益项数
- **长期会员奖励**：
  - 月卡：0分
  - 季卡：15分
  - 半年卡：25分
  - 年卡：50分（最高奖励）

### 月均成本计算

```javascript
月均成本 = 套餐价格 / 有效月数
```

**有效月数映射：**
- 月卡：1个月
- 季卡：3个月
- 半年卡：6个月
- 年卡：12个月

## 💡 使用场景

### 场景1：查看健身房详情

1. 用户点击首页健身房卡片
2. 进入详情页，默认显示"套餐对比"标签
3. 顶部提示"已为您计算最划算套餐"
4. 点击"📊 开启对比模式"
5. 查看所有套餐的详细对比数据
6. 选择"🏆 最划算"套餐

### 场景2：首页快速浏览

1. 用户打开首页
2. 看到"已为您智能筛选附近最划算健身房"提示
3. 每张卡片显示该健身房最划算套餐
4. 第一名带有"🏆 最划算"徽章
5. 点击感兴趣的健身房查看详情

### 场景3：月季半年卡对比

1. 进入详情页
2. 开启对比模式
3. 对比查看：
   - **月卡**：月均成本 = 套餐价格
   - **季卡**：月均成本 = 套餐价格 ÷ 3
   - **半年卡**：月均成本 = 套餐价格 ÷ 6
   - **年卡**：月均成本 = 套餐价格 ÷ 12
4. 根据月均成本选择最适合的套餐
